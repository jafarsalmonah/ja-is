Image Manager

A React-based image management interface focused on performance, predictable data handling, and practical UX decisions. Built as a real-world feature, not a demo toy.

Key Features

Optimistic UI Updates

Images are removed instantly from the UI before the server confirms deletion. Rollback logic ensures consistency on failure.

Virtualized Gallery

The image grid uses windowing to efficiently handle large lists without unnecessary DOM rendering.

Annotation Playground

Users can add and drag bounding boxes over images using a canvas-based approach for better performance and control.

Predictable Data Layer

Server state is handled with React Query, keeping API logic isolated and easy to reason about.

Technology Stack

Next.js React React Query Material UI Axios react-window Konva.js

Design Notes

This project intentionally favors clarity over abstraction. Some logic is kept close together to reduce cognitive overhead while the feature is evolving. Refactoring is planned only when the behavior stabilizes.
